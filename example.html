Huntflow Menu Organizer - Страница для тестирования

Эта страница имитирует структуру Huntflow для тестирования плагина организации меню.

Инструкции по использованию:

  1. Загрузите расширение в режиме разработчика Chrome
  2. Плагин автоматически должен активироваться на этой странице
  3. В левом сайдбаре появится кнопка "Добавить категорию"
  4. Создайте категории и перетаскивайте вакансии между ними
  5. Проверьте сохранение состояния при обновлении страницы

Тестируемые функции:

  • Создание и удаление категорий
  • Переименование категорий
  • Перетаскивание вакансий между категориями
  • Сворачивание/разворачивание категорий
  • Перемещение в "Без категории"
  • Сохранение состояния в Chrome Storage
